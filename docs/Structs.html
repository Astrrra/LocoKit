  • ClassifierResults

    The results of a call to classify(_:types:) on an ActivityTypeClassifier.

    Classifier Results are an iterable sequence of result rows (ClassifierResultItem), with each row representing a single ActivityTypeName and its match probability score.

    The results are ordered from best match to worst match, thus the first result row represents the best match activity type for the given sample. The simplest way to use the results is to take first result row and ignore the rest.

    Interpreting Classifier Results

    The probability scores of each result row range from 0.0 to 1.0, with 0.0 representing 0% match and 1.0 representing a 100% match.
